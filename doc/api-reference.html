    vdev API Reference

    Modules

    Cashtrail

    Cashtrail is an application to track and manage money.

    Cashtrail.Banking

    Cashtrail.Banking keeps the business logic related Currencies, Accounts, and Transactions of the entity, each in their respective contexts.

    Cashtrail.Banking.Currencies

    The Currencies context manages the currencies data of one entity.

    Cashtrail.Banking.Currencies.Currency

    This is an Ecto.Schema struct that represents a currency of the entity.

    Cashtrail.Contacts

    The Contacts context manages the contact data of one entity.

    Cashtrail.Contacts.Address

    This is an Ecto.Schema struct that represents an address of the contact.

    Cashtrail.Contacts.Category

    This is an Ecto.Schema struct that represents a category of one contact and the entity.

    Cashtrail.Contacts.Contact

    This is an Ecto.Schema struct that represents a contact of the entity.

    Cashtrail.Entities

    The Entities context manages the data related to entities. An Entity keeps all financial data of something, that can be a company, financial finances, organization, church, event, etc. And they can have one owner or other members, as well.

    Cashtrail.Entities.Entity

    This is an Ecto.Schema struct that represents an entity of the application.

    Cashtrail.Entities.EntityMember

    This is an Ecto.Schema struct that represents a member of a Cashtrail.Entity that links authorized users to the Cashtrail.Entity, except the owner.

    Cashtrail.Entities.Tenants

    Deals with tenants creation for Entity.

    Cashtrail.Paginator

    Allow fetching paged data with its metadata.

    Cashtrail.QueryBuilder

    This module is responsible to generate queries for contexts to filter and search by schema fields.

    Cashtrail.Repo
    Cashtrail.Users

    The Users context manages the users data of one entity and performs user authentication.

    Cashtrail.Users.User

    This is an Ecto.Schema struct that represents a user of the application.

    CashtrailWeb

    The entrypoint for defining your web interface, such as controllers, views, channels and so on.

    CashtrailWeb.ErrorHelpers

    Conveniences for translating and building error messages.

    CashtrailWeb.Gettext

    A module providing Internationalization with a gettext-based API.

    CashtrailWeb.Router.Helpers

    Module with named helpers generated from CashtrailWeb.Router.
